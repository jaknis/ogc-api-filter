POC – Tillgängliggörande av SGUs OGC APIer


Sidan har tagits fram som en POC för tillgängliggörande av SGUs öppna APIer och förenkla filtrering av data för nedladdning.
Denna version innehåller ett begränsat antal APIer som rör datavärdskapen för grundvatten och miljögifter.
I produktbeskrivningarna för respektive produkt finns en lista över de attribut som kan användas för utsökningen och vad dessa betyder.
Denna applikation kan komma att förbättras, byggas om eller tas bort och skall användas för att utvärdera och ge förslag på förbättringar och nya funktioner.

Nytt i detta skript är att analysdata för grundvatten kan visas i graf.
Välj API:et grundvattenkvalitet-analysresultat och filtrera ut en datamängd. Klicka sedan på Visa diagram under kartan.

Välj vilka fält du vill visa

Avmarkera rutan Visa alla attribut för att välja specifika fält. Urvalet styr vilka fält som visas i tabellen, kartan och Excel-nedladdningen. API-länkarna inkluderar alltid alla fält.

Skapa filter

Här skapar du ett filter för att söka data från OGC-API:et. Filtreringen är skiftlägeskänslig – abc matchar inte Abc.

Operatorn IN används för att matcha en lista av värden, t.ex.:
kommun IN 'UPPSALA','TÄBY','KNIVSTA' eller nationellt_provplatsid IN 116645,310653
IS NULL och IS NOT NULL kan användas för att välja poster som saknar respektive har ett värde.

Obs: Kartan visar bara markörer för data med geometri!
